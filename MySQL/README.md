## Como criar o banco
Para executar o projeto, basta seguir os seguintes passos:

1. Abra o projeto;
2. Execute o script do banco em `MySQL/Scripts.sql`;
3. Popule o banco de dados em `MySQL/Query_Inserts.sql`;
